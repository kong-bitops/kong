기획

농구화 선택 시 리뷰 정보가 분산되어 있고, 레트로 감성과 실전 성능을 동시에 비교하기 어렵다는 문제를 해결하고자 시작했다. 사용자가 자신의 플레이 스타일을 입력하면, 현대형/레트로형 모델을 함께 추천하고 이유를 설명하는 도구를 목표로 했다.

구성물

  • Windows 데스크톱 앱 (Electron + React + TypeScript)
  • 로컬 DB(SQLite) 기반 농구화 데이터 및 즐겨찾기/추천 로그 저장
  • 신발 탐색(필터/검색), 2~3개 모델 비교(차트/표), 맞춤 추천
  • 한국어/영어 전환, CSV 내보내기, 비교 스크린샷 저장
  • 모델 이미지 표시(카드/비교/추천 화면)

실현 방법

  • 프론트엔드: React Router 기반 다중 화면 구성
  • 백엔드(로컬): Electron IPC + better-sqlite3
  • 점수 엔진: Traction, Cushioning, Support, Weight 등 가중치 계산
  • 데이터 규칙: releaseYear 기준으로 자동 분류 (2000년 미만=Retro, 2000년 이상=Modern)
  • AI 도구(Codex): 구조 설계, 코드 생성, 리팩토링, 패키징 자동화

결과

농구화 모델 수를 확장하고, 비교 화면에서 선택 불가 이슈를 해결했다. 신발 탐색 및 비교/추천 화면에서 이미지가 표시되도록 개선했으며, 한국어 UI와 권한 설정까지 포함한 exe 배포 파일 생성이 가능해졌다.

특장점

  • 단순 쇼핑형이 아닌, 설명 가능한 비교/추천 유틸리티
  • 세대(현대/레트로)와 포지션/코트 환경을 함께 반영
  • 로컬 저장 기반으로 네트워크 의존도 낮고 빠른 응답
  • 대회 시연에 유리한 비교표/스크린샷/CSV 내보내기 제공

한계점

  • 실제 착화감은 개인차가 크므로 점수만으로 완전 대체 불가
  • 가격/재고의 실시간 연동은 아직 미구현
  • 모델별 이미지 출처 및 정확도 개선 여지 존재
